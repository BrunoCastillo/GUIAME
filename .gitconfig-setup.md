# Configuración de Git

## Configurar Identidad de Git

Antes de hacer commits, necesitas configurar tu nombre y email en Git.

### Opción 1: Configuración Global (Recomendada)

Configura tu identidad para todos los repositorios:

```bash
git config --global user.name "Tu Nombre Completo"
git config --global user.email "tu-email@ejemplo.com"
```

**Ejemplo:**
```bash
git config --global user.name "Juan Pérez"
git config --global user.email "juan.perez@ejemplo.com"
```

### Opción 2: Configuración Solo para Este Repositorio

Si solo quieres configurarlo para este proyecto:

```bash
git config user.name "Tu Nombre Completo"
git config user.email "tu-email@ejemplo.com"
```

## Verificar Configuración

Para verificar que se configuró correctamente:

```bash
# Ver configuración global
git config --global user.name
git config --global user.email

# Ver configuración del repositorio actual
git config user.name
git config user.email
```

## Usar Email de GitHub

Si tienes cuenta en GitHub, usa el email asociado a tu cuenta:

1. Ve a [GitHub Settings > Emails](https://github.com/settings/emails)
2. Copia tu email (puede ser el privado si lo tienes configurado)
3. Úsalo en el comando:

```bash
git config --global user.email "tu-email@github.com"
```

## Nota sobre Privacidad

GitHub ofrece un email privado con formato: `username@users.noreply.github.com`

Puedes usarlo para mantener tu email privado:

```bash
git config --global user.email "TU_USERNAME@users.noreply.github.com"
```

## Después de Configurar

Una vez configurado, puedes hacer tu commit:

```bash
git add .
git commit -m "Initial commit: Plataforma de Capacitación"
```
